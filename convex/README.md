# Convex backend

This directory contains Siap's normalized data model, custom-JWT
authentication, ownership checks, public application APIs, and
analysis-service mutations.

Before changing it, read:

- `../docs/BACKEND.md`
- `_generated/ai/guidelines.md`

Use `bun run convex:dev` during development and `bun run convex:codegen` to
refresh generated bindings after schema or function changes.
